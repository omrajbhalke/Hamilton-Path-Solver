How to Play the Hamilton Path Game

Objective: Determine if the given graph contains a Hamiltonian path. A Hamiltonian path is a path in a graph that visits every vertex exactly once.

Steps to Play:

  • Choose a Graph: Select a graph by clicking on one of the Graph options.
  • Set Difficulty: Pick the desired difficulty level.
  • Enter the Path: Use the input box to enter the path you believe is a Hamiltonian path. The path should be a sequence of vertices separated by commas (e.g., A,B,C,D).
  • Submit Your Answer: Click the "Submit Solution" button to check if your path is a Hamiltonian path.
  • Get Feedback: The feedback box will inform you if your answer is correct or incorrect. Try again if needed!
  • Try Another Problem: Click the "Try Another Problem" button to get a new graph and start fresh.

Tips:

  • Look for paths that visit every vertex exactly once.
  • Experiment with different graphs and difficulties to improve your understanding of Hamiltonian paths.
  • Use the graph visualization to help you find and verify the path.
